# 验收与 Smoke 清单

## 1. 字幕输入验收

- 空输入能提示错误。
- 普通英文字幕能正常读取。
- 带时间戳字幕能正常读取。
- 超长字幕不会导致程序崩溃。

## 2. 清洗验收

- 多余空行被清理。
- 原文内容没有被改写。
- 时间戳没有误删重要内容。
- 不会把多句话粘成不可读文本。

## 3. 分块验收

- 长字幕能拆成多个 chunk。
- chunk_id 连续。
- 不从单词中间切开。
- 不从句子中间强行切开。
- 每块有前后上下文。

## 4. AI 识别验收

- 返回 JSON 可解析。
- 每条 dialogue 有 speaker_id。
- 每条 dialogue 有 text。
- 不确定人物能标 Unknown Speaker。
- 不编造字幕没有的句子。

## 5. 合并验收

- 分块顺序正确。
- 重叠内容没有重复输出。
- 人物名称基本一致。
- 低置信度人物有标记。

## 6. Markdown 导出验收

- 有人物表。
- 有正文对话。
- 对话顺序正确。
- Markdown 可读。
- 可以复制或下载。

## 7. 失败处理验收

- API 失败有提示。
- JSON 解析失败有提示。
- 单个 chunk 失败不影响其他 chunk。
- 失败块可以重新处理。
